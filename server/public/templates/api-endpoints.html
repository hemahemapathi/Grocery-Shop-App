Grocery Shop API

Welcome to the Grocery Shop API. Use the endpoints below to interact with our services.

Interactive API Documentation Back to Home

Authentication

POST /api/auth/register
Register a new user account
POST /api/auth/login
Login to an existing account
GET /api/auth/profile
Get the current user's profile

Products

GET /api/products
Get all products
GET /api/products/:id
Get a specific product by ID
POST /api/products
Create a new product (admin only)
PUT /api/products/:id
Update a product (admin only)
DELETE /api/products/:id
Delete a product (admin only)

Cart

GET /api/cart
Get the current user's cart
POST /api/cart
Add an item to the cart
PUT /api/cart/:itemId
Update a cart item quantity
DELETE /api/cart/:itemId
Remove an item from the cart

Orders

GET /api/orders
Get all orders for the current user
GET /api/orders/:id
Get a specific order by ID
POST /api/orders
Create a new order
PUT /api/orders/:id
Update an order status (admin only)

Payment

POST /api/create-payment-intent
Create a payment intent for Stripe checkout
